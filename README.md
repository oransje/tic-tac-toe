# Tic-Tac-Toe

A tic-tac-toe cli implementation with Min-Max IA.

## Setup

1. Make a virtualenv and activate it:
```
$ python3 -m venv venv
$ source venv/bin/activate
```

2. Install the dependencies:
```
$ python install -r requirements.txt
```

## Running

Just execute:
```
$ python src/main.py
```
